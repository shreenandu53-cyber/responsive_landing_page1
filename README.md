Responsive Landing Page

Project Overview

This project is a modern and responsive landing page developed using HTML5 and CSS3. It demonstrates the use of Flexbox, responsive web design, hover animations, and modern UI styling to create an attractive and user-friendly webpage.

Features

- Responsive Header with Navigation Menu
- Sticky Navigation Bar
- Modern Hero Section
- Attractive Call-to-Action Button
- Smooth Hover Animations
- Responsive Design using Media Queries
- Feature Cards Section
- Google Fonts (Poppins)
- Modern Color Gradient Background
- Footer with Social Media Links
- Mobile-Friendly Layout
- Clean and Organized Code

Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Google Fonts
- Responsive Web Design

How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click on "index.html".
5. Select Open with Live Server.
6. Resize the browser window to verify the responsive design.
